Overview

This project aims to revolutionize the evaluation process of handwritten answer scripts by leveraging cutting-edge deep learning technologies. By integrating hybrid CNN-LSTM and CNN-GRU models, the system will accurately extract features and classify handwritten answers, ensuring efficiency and fairness in assessments. A user-friendly web application, built with Django, will streamline the evaluation process, enabling seamless interactions for teachers and students alike.
